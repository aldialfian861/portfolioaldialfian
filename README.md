# Portfolio Aldi Alfian

Portfolio satu halaman yang dibangun dengan HTML5, Tailwind CSS CDN, dan JavaScript vanilla. Website ini menampilkan profil, pengalaman, pendidikan, skill, portfolio, galeri, dan kontak dengan nuansa editorial premium.

## Struktur
- index.html
- style.css
- script.js
- assets/images

## Run locally
Buka file index.html di browser atau gunakan server lokal sederhana seperti Live Server.
